For the math part of the evaluation, I used BERTScore with baseline rescaling. I chose this because a regular BERTScore can be misleadingly high - since most models know basic German grammar, they get high points just for sounding human. By "rescaling," I’m basically subtracting the easy points so we can see the actual improvement each model provides.
* The fine-tuned model achieved the strongest overall performance in terms of semantic similarity to the ground truth answers, with a BERTScore F1 of 0.2855, precision of 0.3260 and recall of 0.2494. These results indicate a high degree of alignment with reference answers in both content coverage and phrasing similarity.
* The RAG model achieved an average BERTScore F1 of 0.2180, precision of 0.2685 and recall of 0.1718. While this demonstrates strong semantic similarity, the slightly reduced scores suggest that retrieval constraints and context selection introduce variability in output alignment compared to the fine-tuned model.
* In contrast, the baseline inference model shows substantially lower BERTScore values (F1 = 0.1718, precision = 0.0537 and recall = 0.3087), indicating a conflict in lexical and structural similarity to the reference answers. However, this metric underestimates its actual performance, as shown in the qualitative evaluation.
