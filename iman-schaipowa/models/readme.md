### Overview
Three LLM-based approaches to answering 643 Austrian tax law questions in German, covering EStG, KStG, and UStG etc.

### Models
* Task 1: API Inference, GPT-4o-mini, Local, OpenAI API Key
* Task 2: Fine-Tuning, Llama-3-8B, Google Colab, HuggingFace Key
* Task 3: RAG, Llama-3-8B + MiniLM, Google Colab, HuggingFace Key

### Deliverables
* 3 Jupyter notebooks (inference_model.ipynb, fine_tune_model.ipynb, rag_model.ipynb)
* 3 CSV files (643 answers each; inference_results.csv, fine_tuned_results.csv, rag_results.csv)
* 1 JSON (training data for fine_tune_model.ipynb)
* 1 PDF document (for RAG_model.ipynb)
